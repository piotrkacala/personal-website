# CODEX.md

Repository-level instructions for Codex.

- Converse with the user in Polish.
- Write repository artifacts in English unless the document or feature is explicitly Polish-facing.
- Do not use `Co-authored-by` for AI agents in commit messages.
- If AI assistance should be visible in git history, use explicit trailers such as `Assisted-by: Codex` or `Assisted-by: Claude Code`.
- Only include `Assisted-by` trailers for agents that materially contributed to the changes in that specific commit.
